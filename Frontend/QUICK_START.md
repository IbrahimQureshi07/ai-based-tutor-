## 📋 Step-by-Step Commands (Copy-Paste Ready)

### STEP 1: Dependencies Install
```bash
npm install
```
**Kya Hoga:** Sab packages install hongi (2-3 minutes)

---

### STEP 2: Localhost Pe Test
```bash
npm run dev
```
**Kya Hoga:** 
- Server start hoga
- Browser mein `http://localhost:5173` open karein
- Test karein sab kuch

**Stop:** `Ctrl + C`

---

### STEP 3: Git Commit (Code Save)
```bash
git commit -m "Initial commit: AI-Based Tutor with adaptive learning"
```
**Kya Hoga:** Sab changes save ho jayengi

---

### STEP 4: GitHub Repository Banana

**Pehle GitHub pe jayein:**
1. https://github.com/IbrahimQureshi07 pe jayein
2. "New repository" button click karein
3. Repository name: `ai-based-tutor` (ya kuch bhi)
4. Public ya Private select karein
5. **"Create repository" click karein**

---

### STEP 5: GitHub Pe Push (Agar Repository Banaya Ho)

**Copy-paste karein ye commands (ek ek karke):**

```bash
git remote add origin https://github.com/IbrahimQureshi07/ai-based-tutor.git
```

```bash
git branch -M main
```

```bash
git push -u origin main
```

**Kya Hoga:**
- Code GitHub pe upload ho jayega
- Online available hoga
- Baad mein Vercel/Netlify pe deploy kar sakte hain
